# ANGULAR

## instalar angular en local

1. `npm install @angular/cli@18 --save-dev`
2. `npx ng new angular-ecomm --skip-tests`
3. `cd angular-ecomm`
4. `npx ng g component components/header --skip-tests`
5. `npx ng serve` (lanzar server de dev)
